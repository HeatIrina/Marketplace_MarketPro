# Дашборд операционного мониторинга и аналитики маркетплейса «МаркетПро»

## Описание проекта
Отслеживание ключевых показателей для повышения эффективности продаж, оптимизации ассортимента и удержания пользователей.
 
## Этапы проекта

1. Знакомство с данными и предобработка.
2. Создание датасетов:  
   2.1. Для построения графиков по продажам, категориям товаров, активности продавцов и покупателей.  
   2.2. Для анализа регистраций, активности пользователей, а также для построения графиков Retention и активности по времени.  
3. Создание дашборда.
4. Тестирование дашборда.
5. Руководство по работе с дашбордом.

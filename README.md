# Дашборд операционного мониторинга и аналитики маркетплейса «МаркетПро»

## Описание проекта

Отслеживание ключевых показателей для повышения эффективности продаж, оптимизации ассортимента и удержания пользователей.
 
## Этапы проекта

1. Знакомство с данными и предобработка.
2. Создание датасетов:  
   2.1. Для построения графиков по продажам, категориям товаров, активности продавцов и покупателей.  
   2.2. Для анализа регистраций, активности пользователей, а также для построения графиков Retention и активности по времени.  
3. Создание дашборда.
4. Тестирование дашборда.
5. Руководство по работе с дашбордом.

## Навыки и инструменты

**Навыки**
1. Подготовка данных.
2. Аналитическая проверка.
3. Визуализация и отчётность.

**Инструменты**  
`Jupyter Notebook`, `Python`, `pandas`, `numpy`, `matplotlib`, `pyplot`, `seaborn`.

## Вывод

Дашборд объединил 25 визуализаций по продажам, пользователям и качеству сервиса. Позволяет отслеживать GMV, конверсию, удержание (Retention, LTV, ARPU) и операционно управлять ассортиментом через топ‑категории и статусы заказов. Интерактивные фильтры по датам, категориям и рейтингам дают возможность самостоятельно исследовать причины изменений ключевых метрик.
